# 熔盐工程智能协同工作台

一个用于项目汇报展示的精简版前端 Demo，展示“多智能体 + Skill 工具库 + 工程任务流 + 候选配方评审 + 实验方案 + 报告预览”的产品概念。

## Demo 定位

本项目面向储热、传热、火电调峰与熔盐储能场景，模拟工程人员输入自然语言需求后，多智能体协同调用工程 Skill 工具库，生成候选熔盐体系、市场与招标洞察、实验验证方案和工程推荐报告。

当前版本为纯前端演示，不包含真实后端、真实大模型接口或外部 API Key。所有内容均为 mock 数据。

## 核心功能

- 首页 / 工作台：自然语言需求输入与“开始智能分析”入口。
- Agent 面板：展示 7 个核心 Agent 的运行状态、职责和输出摘要。
- Skill 工具库：展示 8 个工程 Skill，突出 Agent 调用工具完成具体任务。
- StepTimeline：按 0.6 秒节奏推进 8 个工程任务步骤。
- 候选配方评审：展示 3 个 mock 熔盐候选体系及工程建议。
- 市场与招标洞察：展示原盐采购风险和工程招标匹配线索。
- 实验方案：展示选中候选体系的称量表、预干燥、熔融、DSC、TG 与腐蚀实验建议。
- 报告预览：展示工程推荐报告目录，并提供复制摘要和 Demo 导出提示。

## 运行方式

```bash
npm install
npm run dev
```

构建生产版本：

```bash
npm run build
```

本地预览构建结果：

```bash
npm run preview
```

## 技术栈

- Vue 3
- Vite
- TypeScript
- 纯 CSS 视觉实现
- 纯前端 mock 数据

## 后续可接入的真实后端能力

- 大模型任务编排与 Agent 状态流接口。
- 熔盐物性数据库、文献检索与配方相似度检索服务。
- 采购报价、供应商、招投标数据的实时检索与结构化抽取。
- 配方成本估算、腐蚀风险评估和工程可行性校核模型。
- 实验方案生成、实验数据回填与报告自动生成服务。
